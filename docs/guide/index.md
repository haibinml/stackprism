# 使用指南

这一章讲怎么把 StackPrism 装到浏览器里，怎么读懂弹窗里的检测结果，以及附带的几个辅助工具（搜索、原始线索、复制 JSON、纠正反馈）。

## 章节

- [安装与启用](./install.md) — 三种安装方式（Chrome Web Store、Edge 商店、本地加载）
- [基本使用](./basic-usage.md) — 弹窗里五个区域是干什么的
- [结果解读](./results.md) — 置信度、证据、来源、纠正按钮怎么看
- [辅助工具](./tools.md) — 网页源代码搜索、原始线索面板、复制检测 JSON

## 一句话上手

打开任意 `http://` 或 `https://` 网页 → 点击浏览器右上角的 StackPrism 图标 → 弹窗顶部显示「重点 / 全部」分类切换器，下面是技术列表，每条带置信度。

完。如果想进一步定制识别规则、自定义弹窗样式、屏蔽某些技术，移步 [配置指南](/config/)。
